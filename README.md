# dw-form

A form element which is used to get serialized data and to perform validation of it's local elements.

#Installation

npm install @dw/dw-form

## Usage

```html
  <dw-form>
    <dw-input label="Name" required></dw-input>
    <dw-checkbox label="I Agree"></>
  </dw-form>
```

## Methods

- serialized

- validate
